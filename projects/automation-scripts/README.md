## 🛠️ Projects

### 1. Password Generator
A secure command-line tool built in Python to generate random passwords based on custom user preferences.

#### 🚀 Features
- **Custom Length:** Choose how long you want your password to be.
- **Symbols Option:** Toggle the inclusion of special characters (e.g., `!@#$`).
- **Numbers Option:** Toggle the inclusion of digits (e.g., `1234`).

#### 💻 How to Run
1. Make sure you have Python installed.
2. Open your terminal and navigate to the project folder:
   ```bash
   cd projects/automation-scripts
   ```
3. Run the script:
   ```bash
   python password_generator.py
   ```
